```bash

CONFIG_HMM_MIRROR=y           # Heterogeneous Memory Mirror

# SECTION D: THE AUTONOMIC BRAINSTEM (NPU Integration)

CONFIG_DRM_AMDXDNA=m          # XDNA 2 NPU driver (module)
```

**Boot Parameters (Critical):**

```bash

```

- `preempt=full` ‚Äî Forces full preemption for <150¬µs latency (required for NPU thermal control)
- `amd_pstate=active` ‚Äî Hands frequency control to CPPC hardware (microsecond-scale power adjustments)
- `iommu=pt` ‚Äî Passthrough mode for zero-copy CPU/GPU memory sharing

**The Phoenix Protocol:**
